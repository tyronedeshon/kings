# K•I•N•G•S Organization — Upload-Ready (no custom domain)
    
## How to publish
1) Create a new public repo on GitHub (e.g., `kings`).
2) In the repo, click **Add file → Upload files**.
3) Drag *everything from this folder* (index.html, .nojekyll, 404.html, README.md, the assets/ folder) into GitHub and **Commit**.
4) Settings → **Pages** → Source: **Deploy from a branch** (main / root). Save.
5) Your site will appear at: `https://<your-username>.github.io/<repo-name>/`.
